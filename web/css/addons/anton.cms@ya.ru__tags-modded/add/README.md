# [Modded `<cms:add>`](https://github.com/trendoman/Tweakus-Dilectus/tree/main/anton.cms%40ya.ru__tags-modded/add)

Sum unlimited number of values instead of just 2.

```xml
<cms:add '1' '2' '3' '4' />
```
