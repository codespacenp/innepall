# `<cms:metaphone>`

Generate a metaphone[^1].

[^1]: https://www.php.net/manual/en/function.metaphone

```xml
<cms:metaphone 'hello'>
```

## Examples

```xml
<cms:metaphone 'hello'><br>
<cms:metaphone 'incredible'><br>
<cms:metaphone 'programmer'><br>
```
