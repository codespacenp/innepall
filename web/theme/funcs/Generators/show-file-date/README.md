# show-file

Show modification date of a file

```xml
<cms:call 'show-file-date' 'index.php' />
```

## Parameters

* **file**

   URL or absolute path or relative path to website directory.

## Installation

**Use [Autoloading](https://github.com/trendoman/Cms-Fu/tree/master/ADDON-FUNCS-ON-DEMAND.md),**

or, alternatively, manually copy the func code to the page before the calling or embed func as a snippet â€”

```xml
<cms:embed 'funcs/Generators/show-file-date/show-file-date.func' />
```
